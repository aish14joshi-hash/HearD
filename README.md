# 🧏‍♀️ HearD – Gesture to Text & Communication Assistant

HearD is a web-based AI system designed to help bridge communication between **specially-abled individuals** and others. It converts hand gestures into text using a machine learning model and provides additional support features like text-to-gesture visualization and emergency assistance.


## 🚀 Features

* ✋ Real-time Gesture to Text conversion  
* 🔤 Text to Gesture visualization  
* 🚨 Emergency alert system for quick communication  
* 📚 Learn section for gesture understanding  
* 🧠 AI-based prediction using trained model  
* 🌐 Interactive and user-friendly web interface  


## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript  
* **Backend:** Python (Flask)  
* **Machine Learning:** TensorFlow / Keras  
* **Computer Vision:** OpenCV  
* **Hand Tracking:** MediaPipe  
* **Model Format:** SavedModel  


## 🌐 Compatibility

The HearD web application is fully responsive and works across multiple devices and platforms, including:

* 💻 Laptops  
* 🖥️ Desktop computers  
* 📱 Mobile phones  
* 🤖 Android devices  
* 🍎 Apple devices (Mac, iPhone, iPad)
